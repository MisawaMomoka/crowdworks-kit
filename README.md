# 🚀 クラウドワークス案件獲得自動化キット

毎朝寝ている間に案件リサーチ・スコアリング・提案文生成まで全自動！

---

## できること

| 機能 | 説明 |
|------|------|
| ① 案件リサーチ | 指定キーワードで新着案件を自動収集 |
| ② スコアリング | 5項目（時給・継続性・評価・スキル・精神コスト）で20点満点採点 |
| ③ 提案文生成 | 15点以上の案件にAIが専用提案文を自動生成 |

---

## セットアップ手順

### ステップ1：Python のインストール

Python が未インストールの場合は下記からインストール
https://www.python.org/downloads/
（インストール時に「Add Python to PATH」にチェックを入れること）

### ステップ2：ツールのセットアップ

`setup.bat` をダブルクリックして実行
（初回のみ。ライブラリが自動インストールされます）

### ステップ3：Gemini API キーの取得（無料）

1. https://aistudio.google.com/ にアクセス
2. Googleアカウントでログイン
3. 「Get API key」をクリック
4. 「Create API key」で発行
5. 発行されたキーをコピー

### ステップ4：config.yaml の設定

`config.yaml` をメモ帳で開いて、以下を書き換える：

```yaml
crowdworks:
  email: "あなたのCWメールアドレス"
  password: "あなたのCWパスワード"
  keywords:
    - "データ入力"          ← 検索したいキーワードを追加・変更
    - "メール返信代行"

ai:
  provider: "gemini"
  gemini_api_key: "取得したGemini APIキーを貼り付け"
```

### ステップ5：ツールの起動

`start.bat` をダブルクリック
→ ブラウザが自動で開いてツールが使えるようになる

---

## 使い方

1. **サイドバー**でログイン情報・キーワード・APIキーを確認
2. 「① 案件リサーチ＆スコアリング」タブの **「▶️ リサーチ開始」** をクリック
3. 自動でCWにログインして案件を収集・採点（5〜10分）
4. 「② スコア結果一覧」タブで採点結果を確認
5. 「③ 提案文確認・出力」タブで **「全件の提案文を一括生成」** をクリック
6. 提案文を確認・編集して、Excelに保存
7. CWのサイトに貼り付けて送信！

## スコアリング基準（20点満点）

| 項目 | 配点 | 高得点の条件 |
|------|------|-------------|
| 🕐 時給 | 4点 | 推定時給 3000円以上 |
| 🔄 継続性 | 4点 | 継続・長期案件 |
| ⭐ クライアント評価 | 4点 | 評価 4.8以上 |
| 📚 スキル習得度 | 4点 | プログラミング・デザイン等 |
| 😤 精神コスト | 4点 | レッドフラグなし |

**合計15点以上の案件のみ提案文を生成します**

## よくある質問

**Q: Gemini APIは本当に無料ですか？**
A: 1日1,500リクエストまで無料です。毎日使っても月数十円以内に収まります。

**Q: CWのアカウントがBANされませんか？**
A: ページ間のアクセスに待機時間を設けているため、通常の利用と変わりません。ただし短時間の大量アクセスはリスクがあるため、max_pages は3〜5ページ程度を推奨します。

**Q: 提案文の名前の部分はどうすれば？**
A: 生成された提案文の `[お名前]` 部分を自分の名前に書き換えてください。

**Q: CWのサイトが更新されてうまく動かない場合は？**
A: `scraper.py` のセレクタ部分を更新する必要がある場合があります。
