# クラウドワークス案件獲得自動化キット — 完全マニュアル


━━━━━━━━━━━━━━━━━━━━━━━━━━

■ 第1章：はじめに — このツールであなたの案件獲得が変わる

━━━━━━━━━━━━━━━━━━━━━━━━━━

クラウドワークスで稼ぎたい。でも、こんな悩みはありませんか？

・案件が多すぎて、どれに応募すればいいかわからない
・毎日案件を探すのが面倒で続かない
・提案文を書くのに時間がかかる
・そもそも何を書けば採用されるのかわからない

このツールは、その全てを自動化します。

あなたがやることは、たった3つだけ。

1. キーワードを入力する（例：「ライティング」「データ入力」）
2. 「リサーチ開始」ボタンを押す
3. 生成された提案文をコピーしてクラウドワークスに貼る

たったこれだけです。

ツールが裏側でやってくれることはこちらです。

【① リサーチ】
→ 指定キーワードで新着案件を自動収集
→ あなたはボタンを押すだけ

【② スコアリング】
→ 5項目で20点満点で案件を自動採点
→ あなたは結果を見るだけ

【③ 提案文生成】
→ 高スコアの案件にAIが専用の提案文を自動作成
→ あなたはコピペして送信するだけ

案件リサーチに毎日30分〜1時間かけていた作業が、ボタン1つで完了します。
しかも、AIが案件の中身を読んで「その案件専用の提案文」を書いてくれるので、コピペ提案文とは採用率が段違いです。


▼ スコアリングの仕組み

このツールは案件を5つの観点で自動採点します。

【時給（4点満点）】
報酬額から推定時給を計算。時給3,000円以上なら満点

【継続性（4点満点）】
「継続」「長期」「毎月」などのキーワードがあるか

【クライアント評価（4点満点）】
クライアントの過去の評価。4.8以上なら満点

【スキル習得度（4点満点）】
その仕事で新しいスキルが身につくか

【精神コスト（4点満点）】
「修正無制限」「急ぎ」などの地雷ワードがないか

合計20点満点で、設定した閾値（初期設定は15点）以上の案件だけに提案文を自動生成します。
つまり、「割に合わない案件」「地雷案件」を自動で弾いてくれるということです。

▼ STEP 1：Python をインストールする

Pythonとは、このツールを動かすために必要なソフトです。無料でインストールできます。

1. 以下のURLをブラウザで開いてください。
https://www.python.org/downloads/

2. 画面に大きく表示される黄色い「Download Python 3.xx.x」ボタンをクリックしてください。

3. ダウンロードしたファイル（python-3.xx.x-amd64.exe）をダブルクリックして開きます。

4. インストール画面が表示されます。ここが最も重要です。

画面の下の方に「Add Python to PATH」というチェックボックスがあります。

★★★ 超重要 ★★★
「Add Python to PATH」に必ずチェックを入れてください。
これを忘れるとツールが動きません。
もし忘れた場合は、一度アンインストールしてからやり直してください。

チェックを入れたら「Install Now」をクリックしてください。

5. 「Setup was successful」と表示されたら完了です。「Close」を押してください。


▼ STEP 2：ツールのファイルを配置する

1. ダウンロードしたZIPファイルを右クリック →「すべて展開」を選んでください。

2. 展開されたフォルダを、わかりやすい場所に移動します。おすすめはデスクトップです。
（例：デスクトップに「crowdworks_kit」フォルダがある状態）


▼ STEP 3：セットアップを実行する（初回のみ）

1. crowdworks_kit フォルダを開いてください。

2. フォルダ内にある「setup.bat」をダブルクリックしてください。

3. 黒い画面（コマンドプロンプト）が開いて、自動でセットアップが始まります。
画面に以下のようなメッセージが流れます。

---
必要なライブラリをインストールしています...
playwright をインストールしています...
ブラウザをインストールしています...
セットアップが完了しました！
---

4. 「セットアップが完了しました」と表示されたら、何かキーを押して画面を閉じてください。

★ 黒い画面が一瞬で消えてしまう場合：
→ フォルダ内の空白部分で「Shift + 右クリック」→「ターミナルを開く」を選び、
  「setup.bat」と入力してEnterキーを押してください。

★ 赤い文字でエラーが出た場合：
→ ほとんどの場合、STEP 1 で「Add Python to PATH」にチェックを入れ忘れたことが原因です。
  Pythonをアンインストールして、STEP 1 からやり直してください。


▼ STEP 4：Gemini API キーを取得する（無料）

このツールはAIに提案文を書いてもらうため、「APIキー」というものが必要です。
Googleアカウントがあれば無料で取得できます。

1. 以下のURLをブラウザで開いてください。
https://aistudio.google.com/

2. Googleアカウントでログインしてください。

3. 画面左側のメニューにある「Get API key」をクリックしてください。

4. 「Create API key」ボタンをクリックしてください。

5. プロジェクトの選択画面が出たら、そのまま「Create API key in new project」をクリックしてください。

6. APIキーが表示されます。これをコピーしてください。
（例：AIzaSyBxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx）

※ このAPIキーは他人に教えないでください。あなた専用の鍵です。

※ 無料枠について：Gemini APIは1日1,500リクエストまで無料で使えます。
  このツールで日常的に使う分には無料枠を超えることはまずありません。


▼ STEP 5：ツールを起動する

1. crowdworks_kit フォルダ内の「start.bat」をダブルクリックしてください。

2. 黒い画面が開き、数秒後にブラウザが自動で開きます。
黒い画面に以下のようなメッセージが表示されます。

---
You can now view your Streamlit app in your browser.
Local URL: http://localhost:8501
---

3. ブラウザに「クラウドワークス 案件獲得自動化キット」という画面が表示されたら、セットアップ完了です！

★ ブラウザが自動で開かない場合：
→ ブラウザを自分で開いて、アドレスバーに以下を入力してください。
  http://localhost:8501

★ 終了するとき：
→ 黒い画面（コマンドプロンプト）を閉じればツールが終了します。

【ターミナルの開き方】
キーボードで「command + スペース」を同時に押す
→ 検索バーに「ターミナル」と入力
→ Enterキーを押す

cd ~/Desktop/crowdworks_kit && pip3 install -r requirements.txt && python3 -m playwright install chromium

※ デスクトップ以外の場所にフォルダを置いた場合は、~/Desktop/crowdworks_kit の部分をフォルダの場所に変えてください。

3. インストールが始まります。数分かかることがあります。
画面に文字がたくさん流れますが、正常です。そのまま待ってください。

4. エラーなく終わればセットアップ完了です。

★ 「pip3: command not found」と出た場合：
→ 以下のコマンドを先に実行してください。
  python3 -m ensurepip --upgrade
→ その後、STEP 3 のコマンドを再度実行してください。

★ 「Permission denied」と出た場合：
→ コマンドの先頭に sudo を付けて実行してください（パスワードを求められます）。
  sudo pip3 install -r requirements.txt

※ 次回からの起動も同じコマンドです。毎回ターミナルを開いてこのコマンドを実行してください。

★ もっと簡単に起動したい方へ（ショートカットの作り方）：

6. 以降は「start.command」をダブルクリックするだけで起動できます！

★ 終了するとき：
→ ターミナルで Ctrl + C を押すか、ターミナルのウィンドウを閉じてください。

■ 第3章：使い方ガイド

━━━━━━━━━━━━━━━━━━━━━━━━━━


▼ 3-1. 初期設定をする

ツールを起動したら、まず左側のサイドバーで初期設定を行います。


【クラウドワークス ログイン情報】

・メールアドレス → クラウドワークスに登録しているメールアドレス
・パスワード → クラウドワークスのログインパスワード

※ セキュリティについて：入力した情報はあなたのPC内にのみ保存されます。外部に送信されることはありません。ツールがクラウドワークスにログインするためだけに使用します。


【検索キーワード】

案件を検索するキーワードを入力します。1行に1つずつ入力してください。

例：
データ入力
ライティング
メール返信代行
事務

キーワード選びのコツ：
・まずは自分がやりたい・できそうな仕事のジャンルを入力する
・具体的なキーワードの方が良い案件が見つかりやすい
 （「ライティング」より「SEO記事 ライティング」など）
・3〜5個くらいが目安。多すぎると処理に時間がかかります


【各キーワードの取得ページ数】

1キーワードあたり何ページ分の案件を取得するかの設定です。

・推奨：2〜3ページ
・ページ数を増やすと取得件数は増えますが、処理時間も長くなります
・5ページ以上はクラウドワークスへの負荷を考え、非推奨です


【提案文を生成するスコア閾値】

・初期設定：15点（20点満点中）
・15点以上の案件にだけ提案文が自動生成されます
・結果が少なすぎる場合は12〜13点に下げてみてください
・厳選したい場合は17〜18点に上げてください


【AI設定】

・AIプロバイダー → 「Gemini（無料枠あり）」を選んでください
・APIキー → STEP 4 で取得したGemini APIキーを貼り付けてください


【あなたのプロフィール】

ここに自分のスキルや経験を入力しておくと、提案文にあなたの情報が自然に盛り込まれます。

例：
Webデザイン歴5年。LP・バナー制作を中心に活動しています。
WordPressのカスタマイズも対応可能。レスポンシブデザインが得意です。

※ まだプロフィールが書けていない方は、第6章で紹介する「CWプロフィールアシスタント（GPTs）」を使えば、質問に答えるだけでプロフィール文が完成します。


▼ 3-2. 案件リサーチを実行する

1. 「リサーチ」タブの「リサーチ開始」ボタンをクリックしてください。

2. ツールが自動で以下の処理を行います（5〜10分かかります）。

---
クラウドワークスにログイン中...
キーワード「データ入力」を検索中...
キーワード「ライティング」を検索中...
42 件を取得しました。スコアリング中...
完了。42 件取得 / スコア 15 点以上: 8 件が提案文生成対象です。
---

3. 処理が完了したら「スコア結果」タブで結果を確認できます。

※ 処理中はブラウザを閉じないでください。黒い画面（ターミナル）も閉じないでください。


▼ 3-3. スコア結果を確認する

「スコア結果」タブを開くと、取得した案件の一覧が表示されます。

【表示フィルターの切り替え】
・全件 → 取得した全ての案件を表示
・合格のみ → 閾値以上のスコアの案件だけ表示
・除外のみ → 閾値未満の案件だけ表示

【結果の見方】

各案件をクリックすると詳細が開きます。以下は表示例です。

---
【合格】 18/20点  SEO記事のライティング（3000文字×10記事）

案件URL: https://crowdworks.jp/public/jobs/xxxxx
報酬: 30,000〜50,000円
継続: 継続案件
クライアント評価: 4.9

スコア内訳
時給      4/4 — 推定時給 3,000円 (3000円以上)
継続性    4/4 — 継続・長期案件
CL評価    4/4 — 評価 4.9 (優良)
スキル    3/4 — 中スキル (ライター, 記事)
精神      3/4 — 注意点あり (即日)
合計: 18/20点
---

※ スコアはあくまで目安です。気になる案件はURLをクリックして、クラウドワークスで実際の募集要項を確認してください。


▼ 3-4. 提案文を生成する

1. 「提案文」タブを開いてください。

2. 「全○件の提案文を一括生成」ボタンをクリックしてください。

3. AIが1件ずつ提案文を生成します（1件あたり5〜15秒程度）。

4. 生成が完了すると、各案件の提案文が表示されます。


【提案文の確認・編集】

生成された提案文はそのまま使えますが、以下の点を確認してください。

・「[お名前]」の部分を自分の名前に書き換える
・内容に違和感がないか読み返す
・自分の経験に合わない部分があれば修正する

※ 提案文はテキストエリア内で直接編集できます。
※ 内容が気に入らない場合は「再生成」ボタンで書き直すこともできます。


【クラウドワークスに提案を送る手順】

1. 提案文の下にある案件URLをクリックして、クラウドワークスの案件ページを開きます。
2. 「応募画面へ」ボタンをクリックします。
3. 提案文をコピーして、メッセージ欄に貼り付けます。
4. 金額や納期を入力して送信します。


▼ 3-5. 結果をExcelで保存する

「Excelでダウンロード」ボタンを押すと、全ての結果（スコア・提案文含む）をExcelファイルで保存できます。

・提案文なしの結果だけ保存したい場合 → 「スコア結果」タブのダウンロードボタン
・提案文込みで保存したい場合 → 「提案文」タブのダウンロードボタン

■ 第4章：効果を最大化するコツ

━━━━━━━━━━━━━━━━━━━━━━━━━━


▼ コツ1：キーワードは「やりたいこと」＋「具体的なスキル名」で

悪い例：仕事、副業、在宅
良い例：SEO記事 ライティング、WordPress カスタマイズ、Instagram運用代行

具体的なキーワードの方が、自分に合った案件がヒットしやすくなります。


▼ コツ2：閾値は最初15点、慣れたら調整

・始めたばかりの頃は12〜13点にして、多めに案件を見る
・慣れてきたら15〜17点に上げて、良い案件だけに集中する


▼ コツ3：提案文は必ず「自分の言葉」を足す

AIが生成した提案文はそのまま使えますが、以下を足すとさらに採用率が上がります。

・その案件に興味を持った理由
・自分の具体的な経験やエピソード
・「いつから作業開始できるか」の明記


▼ コツ4：毎日〜2日に1回のペースでリサーチ

クラウドワークスは毎日新しい案件が投稿されます。週に3〜4回リサーチを回すのがおすすめです。
所要時間は1回あたり5〜10分です。


▼ コツ5：プロフィール欄を充実させる

サイドバーの「あなたのプロフィール」に情報を入れておくと、提案文に自然にあなたの強みが盛り込まれます。
プロフィールが充実しているほど、AIが書く提案文の質も上がります。

Q：setup.bat をダブルクリックしても何も起きません

A：以下を試してください。
1. フォルダ内の空白部分で「Shift + 右クリック」→「ターミナルを開く」
2. 「setup.bat」と入力してEnter

A：Pythonインストール時に「Add Python to PATH」にチェックを入れ忘れた可能性が高いです。
Pythonをアンインストールしてから、チェックを入れて再インストールしてください。


Q：Macでターミナルのコマンドがエラーになります

A：以下を順番に試してください。
1. 「python3 --version」と入力して、Pythonがインストールされているか確認
2. インストールされていない場合は STEP 1 からやり直す
3. 「pip3」が使えない場合は以下を試す
   python3 -m pip install -r requirements.txt


【ツールの動作関連】


Q：リサーチ開始を押しても案件が0件です

A：以下を確認してください。
1. クラウドワークスのメールアドレスとパスワードが正しいか
2. キーワードが適切か（あまりにニッチだと結果が0件になります）
3. インターネットに接続されているか


Q：「ログインに失敗しました」と表示されます

A：以下を確認してください。
1. メールアドレスとパスワードが正しいか
  （ブラウザから手動でログインできるか確認してみてください）
2. クラウドワークスで二段階認証を設定している場合は、一時的にOFFにしてください
3. CAPTCHAが表示されている可能性があります。ブラウザから一度手動でログインしてからツールを再実行してください


Q：提案文が生成されません / APIエラーが出ます

A：以下を確認してください。
1. APIキーが正しくコピーされているか（前後に空白が入っていないか）
2. Gemini API の無料枠を使い切っていないか（翌日にリセットされます）
3. 別のAIプロバイダー（Claude / OpenAI）を試してみてください


Q：処理に時間がかかりすぎます

A：以下を試してください。
1. 取得ページ数を2〜3ページに減らす
2. キーワードの数を3〜4個に絞る
3. インターネット回線の速度を確認する


【費用関連】


Q：Gemini APIは本当に無料ですか？

A：はい。1日あたり1,500リクエストまで無料です。このツールでは1案件につき1リクエストしか使わないので、1日に100件以上の提案文を生成しても無料枠内に収まります。万が一超えても月数十円程度です。


Q：クラウドワークスのアカウントがBANされませんか？

A：ツールはページ間のアクセスに待機時間を設けており、通常のブラウザ操作と変わらない速度でアクセスしています。ただし、安全のために取得ページ数は3〜5ページ程度にしてください。

▼ 6-1. CWプロフィールアシスタント
━━━━━━━━━━━━━━━━━━

【何ができるか】
質問に答えていくだけで、クラウドワークスのプロフィール欄にそのまま使える自己紹介文が完成します。

【こんな方におすすめ】
・プロフィールに何を書けばいいかわからない
・実績がゼロで書くことがない
・文章を書くのが苦手

【使い方】

1. 以下のリンクからGPTsを開いてください。
（※リンクをここに挿入）

2. チャットが開くと、アシスタントが挨拶してくれます。

「はじめまして。このGPTはあなたのクラウドワークスの
 プロフィール文を一緒に作るアシスタントです。
 いくつか質問に答えていただくだけで、プロフィール文が完成します。
 実績がなくても大丈夫です。では始めましょう。」

3. 以下のような質問が1つずつ出てきます。思いつくまま答えてください。

【質問①】これまでどんな仕事や作業をしてきましたか？
→ 仕事でなくてもOK。家事・育児・趣味・学校の活動なども立派な経験です

【質問②】得意なことや褒められたことは？
→ 「丁寧」「早い」「正確」など、何でもOK

【質問③】使えるツールやアプリは？

【質問④】どんな仕事をしてみたいですか？
→ やってみたいこと・興味があることを素直に

【質問⑤】仕事で大切にしていることは？
→ 「納期を守る」「こまめに連絡する」など

【質問⑥】稼働できる時間帯は？
→ 「平日夜2時間」「土日終日」など

4. 全ての質問に答えると、300〜500文字のプロフィール文が生成されます。

5. 生成されたプロフィール文をコピーして、クラウドワークスのプロフィールページに貼り付けてください。

★ ポイント：このプロフィール文は、次に紹介する「CWプロ案件アシスタント」でも使います。コピーして保存しておいてください。

▼ 6-2. CWプロ案件アシスタント
━━━━━━━━━━━━━━━━━━

【何ができるか】
案件の募集要項を貼り付けるだけで、自動でスコアリング（20点満点採点）と提案文の生成を行います。

【こんな方におすすめ】
・良い案件かどうかの判断ができない
・提案文を書くのが苦手・時間がかかる
・PCがなくてもスマホで案件獲得の質を上げたい

【使い方】

1. 以下のリンクからGPTsを開いてください。
（※リンクをここに挿入）

2. クラウドワークスで気になる案件を見つけたら、その募集要項のテキストをコピーしてください。

3. コピーしたテキストをGPTsのチャットに貼り付けて送信してください。

4. まず、プロフィール情報を聞かれます。

「ありがとうございます。提案文をあなたに合わせて作成するため、
 簡単なプロフィールを教えてください。
 ・得意なスキルや経験
 ・これまでの実績（なければ「なし」でOKです）
 ・アピールしたい強み」

★ CWプロフィールアシスタントで作ったプロフィール文をそのまま貼り付ければOKです！

5. プロフィールを送ると、案件のスコアリング結果が表示されます。

---
スコア結果

時給        3/4  時給換算1,500円
継続性      4/4  長期案件
CL評価      3/4  評価4.2
スキル習得度 4/4  新スキル習得可
精神コスト   4/4  問題なし

合計: 18/20  → 合格
---

6. 15点以上の場合は自動で提案文が生成されます。

7. 生成された提案文の「[お名前]」を自分の名前に変えて、クラウドワークスの応募画面に貼り付けてください。

★ 14点以下の場合：「見送りをおすすめします」と表示されますが、それでも提案文を作りたい場合は「作成してください」と返信すれば生成してくれます。

・PC版 → あり
・GPTs → なし

【操作方法】
・PC版 → ボタンをクリック
・GPTs → チャットで対話

【必要なもの】
・PC版 → PC + Gemini APIキー
・GPTs → スマホ + ChatGPTアカウント

★ PC版とGPTsは併用できます。PC版で案件を一括リサーチ → 気になる案件をGPTsで深掘り、という使い方もおすすめです。
